Alt+F4 at any point to stop program

Configure the program in the 'input' folder
Add tail number as needed in the "DO NOT RENAME" sheet
Configure Google Drive information as needed in the 
"Google Drive Config" sheet
Then run program using main.xaml

Downloaded files will be saved to 'Output\PTMD' folder
Then the files will be consolidated and placed in 'Output' folder with the date
Once uploaded to Google Drive, consolidated excel files will be moved to
the "Output\archive" folder

This version does not keep cell formatting such as highlights
